# Tourism Chatbot 🌍

Un chatbot intelligent pour explorer les merveilles touristiques du Maroc. Ce projet utilise une base de données locale pour fournir des informations détaillées sur différentes villes, monuments, et événements.

## 🚀 Fonctionnalités
- **Recherche intelligente** : Trouvez des informations sur des lieux spécifiques.
- **Multilingue** : Support du Français, Anglais et Arabe.
- **Base de données riche** : Informations extraites sur Marrakech, Casablanca, Fès, et plus encore.
- **Interface Web** : Chat interactif moderne.

## 🛠️ Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/MOHSINEDAHNAOUI/tourism-chatbot.git
   ```
2. Installez les dépendances :
   ```bash
   pip install -r requirements.txt
   ```
3. Lancez l'application :
   ```bash
   python app.py
   ```

## 📂 Structure du Projet
- `app.py` : Serveur Flask/Backend.
- `database.json` : Base de données des lieux.
- `index.html` & `script.js` : Frontend du chatbot.
- `scraper.py` : Outil de collecte de données.
